# Base64 Universal Decoder

Choose your preferred language:

- [English](README-en.md)
- [中文](README-zh.md)

Base64 Universal Decoder is a lightweight Manifest V3 Chrome extension for converting images to Base64 and decoding Base64 content into images, text, documents, media, archives, or binary files.

Base64 Universal Decoder 是一个轻量的 Manifest V3 Chrome 扩展，支持图片转 Base64，并可将 Base64 解码为图片、文本、文档、媒体、压缩包或二进制文件。
